# Algorithm Visualizer – DSA Educational Project

Algorithm Visualizer is an **interactive educational tool** developed as part of the **DSA** course in the **3rd Semester**. It secured **1st position** in semester project exhibition.

This project is shared here as a **reference resource** to help juniors:
- Understand algorithm dry-runs visually
- Build intuition for some DSA concepts like Sorting, Backtracking, BST, AVL structures etc.
- Get inspiration for academic and mini projects
